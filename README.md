## 🏭 About

A comprehensive **Occupational Health & Safety (OHS/ISG) Management Portal** built for corporate environments. Digitizes all OHS processes into a single, secure web platform actively used in a production environment.

## ✨ Features

- 📋 **Announcement Management** — Create and publish ISG announcements with image support
- 📨 **Internal Messaging** — Staff-to-staff messaging system
- 🖊️ **Digital Signatures** — Document acknowledgment and e-signature tracking
- 👥 **Role-Based Access** — Admin and employee roles with separate dashboards
- 📊 **Reporting** — ISG activity logs and compliance reports
- 🔔 **Notifications** — System alerts for critical safety updates

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | PHP 8.0+ |
| Database | MySQL 8.0 |
| Frontend | HTML5, CSS3, JavaScript |
| Auth | Session-based with RBAC |

## 📋 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ErenYALAZ/isg_system.git
   ```
2. **Import the database schema** from `database_schema.sql`
3. **Rename** `secrets.php.example` → `secrets.php` and fill in your credentials
4. **Deploy** to your web server (`htdocs` or `www`)

## 🔒 Security

Sensitive credentials are excluded via `.gitignore`. Use the provided `secrets.php.example` template.
